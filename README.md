# Homes & Beyond – Property Mini-Listings

A Vue.js single-page application for browsing short-term rental properties in Cape Town.

## Screenshot

![App Screenshot](./screenshot.png)

## Project Overview

This app displays a curated list of rental properties with the ability to search by title or location, sort by price, and bookmark favourite listings. Unavailable properties are clearly marked with a "Not Available" ribbon.

## Features

- Responsive grid-based property cards
- Live search filtering by title or location
- Price sorting (low to high / high to low)
- "Not Available" ribbon on unavailable listings
- Bookmark / save favourite properties
- Dynamic property count in the header

## Tech Stack

- Vue 3 (Options API)
- Vite
- CSS Grid

## Installation & Running the Project

1. Clone the repository:
   git clone https://github.com/YOUR-USERNAME/property-mini-listings.git

2. Navigate into the project folder:
   cd property-mini-listings

3. Install dependencies:
   npm install

4. Run the development server:
   npm run dev

5. Open your browser at:
   http://localhost:5173
